- Criar conta no Github (https://github.com/), caso não tenha
- Instalar o Git (https://git-scm.com/install/windows) 
- Configurar Git (git config) *Obs: somente se a maquina for  (https://git-scm.com/book/pt-br/v2/Come%c3%a7ando-Configura%c3%a7%c3%a3o-Inicial-do-Git)
- Instalar o Java JDK >=21 (https://www.oracle.com/br/java/technologies/downloads/#java21) e configurar JAVA_HOME
- Instalar o Apache Maven (https://maven.apache.org/download.cgi) e adicionar o caminho do bin nas variáveis de ambiente
- Instalar o VSCode (https://code.visualstudio.com/)
- Instalar extensão Extension Pack for Java (https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack) no VSCode.
- Teste do ambiente de desenvolvimento


|Criar projeto Java no VSCode 
|Utilizando com ferramenta de build o Maven
|Sem Archetype
|Com group ID como com.[seunome] (Ex: com.ramyres)
|Como nome serializejson
|Adicione a dependência gson no arquivo pom.xml (https://mvnrepository.com/artifact/com.google.code.gson/gson/2.13.2)
|Escreva uma classe Pessoa(string name, int idade)
|Na classe Main no método main adicione a serialização utilizando o gson.
|Adicione na raiz do projeto um arquivo README.md e adicione estas instruções.
|Versione o projeto no Github, criando um repositório público, e
